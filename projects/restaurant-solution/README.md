# Restaurant Solution

Full-stack solution for restaurant management.

## Features

### Web App (Admin/Staff)
- [ ] Dashboard with metrics
- [ ] Reservation management
- [ ] Table management
- [ ] Order system
- [ ] Kitchen display (KDS)
- [ ] Menu management
- [ ] Employee management
- [ ] Reports and statistics

### Mobile App (Customers)
- [ ] Digital menu
- [ ] Table ordering (QR)
- [ ] Online reservations
- [ ] Loyalty program
- [ ] Push notifications
- [ ] Order history

## Tech Stack
- Frontend: React / Next.js
- Mobile: React Native
- Backend: Go
- Database: PostgreSQL
- Real-time: WebSockets
