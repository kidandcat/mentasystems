# Plataforma Alquileres Policia

Airbnb para miembros de cuerpos de seguridad en Espana.

## Oportunidad

- **Contacto:** Persona individual con bajo presupuesto
- **Activo:** Gestiona grupo de Telegram con **60.000 miembros**
- **Uso actual:** Comparten casas/alojamientos entre policias via Telegram
- **Oportunidad:** Crear plataforma formal, monetizar con comisiones

## Por que es interesante

- 60k usuarios = demanda validada + distribucion garantizada
- Comunidad de confianza (todos son policias verificados)
- Nicho sin competencia directa
- Potencial de ingresos recurrentes

## Modelos de partnership propuestos

| Modelo | Nosotros recibimos | Riesgo |
|--------|-------------------|--------|
| **Equity partner** | 40-50% de la empresa | Alto riesgo, alta recompensa |
| **Revenue share** | 2-3% de cada transaccion | Menor ownership, ingreso pasivo |
| **Hibrido** | €5-8k upfront + 20-30% equity | Cubre costes + upside |
| **Tech co-founder** | 50/50, nosotros build, el trae usuarios | Partnership real |

## Recomendacion

Pedir **40-50% equity** o estatus de **tech co-founder**. No es solo construir una app - es construir un negocio juntos. El grupo de Telegram es su aportacion, la plataforma es la nuestra.

## Puntos a discutir con el

1. Quien es el propietario? (Crear SL juntos?)
2. Modelo de ingresos: comision por reserva (3-5%?) o suscripcion?
3. Quien gestiona soporte, legal, moderacion?
4. Estrategia de salida si uno quiere salir?

## Modelo de negocio potencial

### Revenue streams
- Comision por reserva: 3-5% al anfitrion + 3-5% al huesped
- Verificacion premium / destacados
- Seguros de alquiler (partnership con aseguradora)

### Costes
- Hosting: ~€50-100/mes inicial
- Pasarela de pagos: ~2% Stripe
- Legal/fiscal: a determinar

### Proyeccion (ejemplo)
- 60k miembros, 5% activos = 3.000 usuarios activos
- 500 reservas/mes x €100 media x 6% comision = €3.000/mes
- Potencial crecimiento si se expande a otros cuerpos (Guardia Civil, bomberos, sanitarios...)
