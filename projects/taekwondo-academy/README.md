# Taekwondo Academy Website

Professional landing page with CMS for a martial arts academy.

## Features

### Public Site
- [ ] Hero with academy info
- [ ] Class schedules
- [ ] Belt/level information
- [ ] Photo/video gallery
- [ ] News and events
- [ ] Instructor team
- [ ] Pricing and enrollment
- [ ] Contact/registration form
- [ ] Location and map

### CMS (Admin Panel)
- [ ] Edit page content
- [ ] Manage schedules
- [ ] Publish news
- [ ] Upload gallery photos
- [ ] Manage received forms
- [ ] Basic SEO

### Extras
- [ ] Responsive (mobile, tablet, desktop)
- [ ] Optimized for local SEO
- [ ] Optimized loading speed
- [ ] SSL included

## Tech Stack
- Frontend: Next.js (static generation)
- CMS: Headless CMS (Strapi / Payload / custom)
- Hosting: Fly.io / Vercel
- Forms: FormSubmit / custom
